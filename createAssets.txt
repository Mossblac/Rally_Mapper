how to convert .png to .svg using inkscape:

1. open File

2. trace bitmap, should be attached to side. 

3. select .png, preview will appear

4. under trace bitmap : single scan - autotrace

5. select .png under the trace (you know you have it if the preview re-appears), delete

6. click fill and stroke window (same side panel as trace bitmap)

7. click on the trace and the fill window will open options (if entire image not selected, path->union, then select again), select color

8. save as .svg
